Linear Search

Linear Search is a very simple and easy to learn algorithm. It's asymptotic time notation is O(n) however that is very slow compared to that of Binary Seacrh.

How it works

It works by checking all values of the array to see if they are equal to the one that we are looking for. It starts checking from the very first element in the array and if that value is different than the one we are looking for then it checks the one after that, and it continues this process until it finds the value that we are seeking or it reaches the end of the array.

Javascript Implementation

The code needed to write this algorithm in Javascript is the following:

//this function returns the index at which the key is found but if it's not then it returns -1
function linear(array, key) {
  for (var i = 0; i < array.length; i++) {
    if (array[i] == key) {
      return i;
    }
  }
  return -1;
}

Binary Search

Binary Search is a simple yet fast searching algorithm with an asymptotic time notation of O(logn). However it is only performed on sorted arrays.

How it works

Binary Search takes advantage of the fact that the array is sorted so it can only be executed on a sorted array. It starts in the middle of the array, checks if it has found the value it's looking for, if not then it checks if the value is bigger or smaller and goes to the left or right part of the array accordingly. Then it continues to do the same process on the other half of the array until it either finds the value it wants or there are no more elements to check in the array.

Javascript Implementation

The code needed to write this algorithm in Javascript is the following:

//this works for arrays sorted in ascending order
function binary(array, key) {
  var start = 0;
  var end = array.length;
  while(start <= end) {
    var mid = Math.floor((start+end)/2);
    if (array[mid] == key) {
      return mid;
    }
    else if (array[mid] > key) {
      end = mid-1;
    }
    else {
      start = mid+1;
    }
  }
  return -1;
}

Bubble Sort

Bubble sort is very simple and easy to learn but slow sorting algorithm. It's definitely one of the best sorting algorithms to learn first.

How it works

Bubble Sort works by repeatedly swapping adjacents elements only if they are in wrong order.

Javascript Implementation

The following code is Bubble Sort written in Javascript:

var Bubble = function(array) {
 var end = array.length; //this is how to know when to stop the for loop and do while loop
 var swapped = false; // a variable used to exit the do while loop
 do {
  for (var i = 0; end > i-1; i++) {
    //if the current element is bigget than the next one then swap
    if (array[i] > array[i+1]) {
      var temp = array[i];
      array[i] = array[i+1];
      array[i+1] = temp;
      swapped = true;
    }
  }
  end--; //because the last element gets sorted
  } while (swapped && end >= 0);
}

Selection Sort

Selection Sort is a very simple and easy to learn sorting algorithm but is very slow when trying to sort very large arrays. However, it is faster than the bubble sort algorithm and it also is a great first sorting algorithm to learn.

How it works

Selection Sort works by repeatedly finding the minimum element in the unsorted part of the array and placing it at the beggining.

Javascript implementation

The following code is Selection Sort written in Javascript:

var Selection = function(array) {
 var mn; //mn means minimum
 var limit = array.length;
 for (var i = 0; array.length>i; i++) {
   mn = i;
   //find the index of the minimum element in the unsorted array
   for (var j = mn+1; limit > j; j++) {
     if (array[mn] > array[j]) {
       mn = j;
     }
   }
   //swap the minimum element in the unsorted array with the first one
   var temp = array[mn];
   array[mn] = array[i];
   array[i] = temp;
 }
}

Insertion Sort

Insertion Sort is a simple and easy to learn sorting algorithm and is very similar to Selection Sort because they both split the array in an unsorted and sorted part. However, Insertion Sort is faster in most cases.

How it works

Insertion sort works by having a sorted and unsorted part in the array. It considers the sorted array to be the first element of the array and the rest to be the unsorted one. Then it repeatedly inserts the first element from the unsorted array into the sorted array and with a quick scan over the sorted array figures out where to place it. It continues with the same logic until there are no more elements left in the unsorted list.

It can be improved by using binary search each time you want to insert an element into the unsorted part of the list, however we are not going to be discussing that here.

Javascript Implementation

var Insertion = function(array) {
 for (var i = 1; array.length > i; i++) {
   var value = array[i];
   for (var j = i-1; j >= 0 && array[j] > value; j--) {
     //swap
     var temp = array[j+1];
     array[j+1] = array[j];
     array[j] = temp;
   }
 }
}

Merge Sort

Merge Sort is a complex and difficult to learn sorting algorithm, however it's very fast.

How it works

Merge Sort divides the array into n subarrays each containing only one element. Then it merges all the subarrays creating a new sorted array.

Javascript Implementation

The following code is Merge Sort written in Javascript

function merge(array, p, q, r) {
  var lowHalf = [];
  var highHalf = [];
  var k = p;
  var i;
  var j;
  //first part of the array
  for (i = 0; k <= q; i++, k++) {
    lowHalf[i] = array[k];
  }
  //second part of the array
  for (j = 0; k <= r; j++, k++) {
    highHalf[j] = array[k];
  }
  k = p;
  i = 0;
  j = 0;
  //merge the two arrays
  while(i < lowHalf.length && j < highHalf.length) {
    if (lowHalf[i] < highHalf[j]) {
      array[k] = lowHalf[i++];
    }
    else {
      array[k] = highHalf[j++];
    }
    k++;
  }
  while (i < lowHalf.length) {
    array[k++] = lowHalf[i++];
  }
  while(j < highHalf.length) {
    array[k++] = highHalf[j++];
  }
};

function mergesort(array, p, r) {
  if (p < r) {
    var q = Math.floor((p+r)/2);
    mergesort(array, p, q);
    mergesort(array, q+1, r);
    merge(array, p, q, r);
  }
};

Quick Sort

Quick Sort is a complex and difficult to learn sorting algorithm, however, it is one of the fastest and most efficient ways to sort arrays and uses a very important technic in computer programming, which is called divide and conquer.

How it works

Quick Sort works by picking a pivot and partitioning the rest of the elements in the array into two subarrays, one subarray contains all the elements in the array which are smaller than the pivot and the other one contains all the elements which are bigger than the pivot. Then, it recursively sorts the subarrays.

There are many different ways to select a pivot. Some of the most common ones are:

  • Pick last element as pivot.
  • Pick first element as pivot.
  • Pick random element as pivot.
  • Pick median(of first, middle and last element in the array) as pivot.

All of them have their advantages and disadvantages, however, here we are going to be picking the last element as pivot.

The most important part in this algorithm is partitioning. The purpose of the partition() function is to divide the array into two subarrays, one containing all elements in the array smaller than the pivot and the other one containing all elements bigger than it. We will be using linear time partitioning which is the best way to partition the array.

Javascript Implementation

Here is the code needed for the partition() function in Javascript:

var partition = function(array, p, r) {
 var q = p;
 for (var j = p; r > j; j++) {
   if (array[r] >= array[j]) {
     //swap the elements
     var temp = array[j];
     array[j] = array[q];
     array[q] = temp;
     q++;
   }
 }
 //swap elements
 var temp = array[r];
 array[r] = array[q];
 array[q] = temp;
 //return the index of the pivot
 return q;
}

Here is the code needed for the quicksort() function in Javascript:

//l is the left limit and r is the right one
var quicksort = function(array, l, r) {
 if (l < r) {
   var p = partition(array, l , r); //this is the pivot
   quicksort(array, l, p-1);
   quicksort(array, p+1, r);
 }
}

Simple Button

This button has some simple parameters so you can create a simple button without getting confused.

How it works

This simple button is an object. The object has the following paramers in order (x, y, w, h, Text, action, Color), x is the x position of the button, y is the y position of the button, w is the width of the button, h is its height, Text is the text that is going to be displayed on top of the button, action is what the button should do when it's clicked on(it must be a function), Color is the color of the button. To display the button just use the command <button-name>.display();(<button-name> is the name you have given to the button, it could be anything)

The code

This is the code of the button object in Javascript:

//x == x position of the button, y == y position of the button, w == width of the button, h == height of the button, Text == the text that you want to be displayed, action == what the button should do when it's clicked on(must be a function), Color == the color of the button
var Button = function(x, y, w, h, Text, action, Color) {
  //variables {
  this.x = x; //the x coordinates
  this.y = y; //the y coordinates
  this.w = w; //the width
  this.h = h; //the height
  this.action = action; //action is going to be a function that gets run when the user clicks on the button
  this.r = 3; //the radius of the button
  this.Text = Text; //the text that the button has
  this.color = Color;
  this.size = 15;
  //}

  //the object's methods {
  //check if the something at the given coordiantes is inside the object
  this.isInside = function(x, y) {
    //x and y are the cooridnates that I want to check if they are inside the object
    return this.x < x && this.x+this.w > x && this.y < y && this.y+this.h > y;
  };
  //display the button
  this.display = function() {
    if (this.isInside(mouseX, mouseY) && mousePressed) {
      //if clicked on button
      this.action();
    }
    else if (this.isInside(mouseX, mouseY)) {
      //if above button
      this.r = 8;
    }
    else {
      //else
      this.r = 3;
    }
    fill(this.color);
    stroke(0, 0, 0);
    rect(this.x, this.y, this.w, this.h, this.r); //show the button
    fill(0, 0, 0); //color for the text
    textSize(this.size);
    text(this.Text, this.x+((this.w-textWidth(this.Text))/2), this.size+this.y+((this.h-this.size)/2));//show the text
  };
  //}
};

Vigenere Cipher

Vigenere Cipher is hard to crack without a key but not impossible.

How it works

The Vigenere Cipher works by having a keyword and using that keyword to shift the text. That happens by shifting the first character by the first character of the keyword, then the second one with the second character of the keyword, and that continues to happen until the keyword has run out of letters. Then it starts from the first letter of the keyword but from the place where it left of in the original text so it can continue with the encryption.

Code

The code for Vigenere Cipher:

function removeSpaces(shiftWordEl) {
  var k = 0;
  var wholeKey = "";

    if (shiftWordEl[k] !== " ") {
      wholeKey += shiftWordEl[k];
    }
    k++;
  }
  return wholeKey;
}

Radial Gradient

Using radial gradient on the background makes the background really attracting if the right combination of colors is used.

How it works

I have made 2 different functions to make the radial gradient. They both do the same thing however one of them is faster and easier to make whereas the other is a bit more customizable.

The first one is called radialBackground() and and has 3 parameters, Color1, Color2 and distance. Color1 is the color that starts in the middle, Color2 is the color that is most evident around the edges and distance is basically how noticeable you want the middle color to be. This function works by adjusting the color of every individual pixel on the screen.

The second one is called radialBackground2() and has 2 parameters, Color1, Color2. Color1 is the color that starts in the center of the screen whereas Color2 is the color that is most noticeable around the edges of the canvas. This function works by drawing ellipses. It starts with one that covers the whole screen and is colored with Color2 and then continiusly makes more ellipses each one being 1 pixel smaller that the previous one and being colored with a color closer to the one in the middle, this is achieved by using lerpColor().

The second function is faster than the first one, however, it doesn't have that third parameter. It's up to you to choose which one is more suitable for your project.

Javascript Implementation

I made two different functions for the radial background. You can use wichever one you want. This is the code needed to write both of them in Javascript:

size(500, 500);

//this one changes the individual color of every pixel
function radialBackground(Color1, Color2, distance) {
  loadPixels();
  var pointX = width/2;
  var pointY = height/2;
  for (var y = 0; y<height; y++) {
    for (var x = 0; x<width; x++) {
      var d = dist(pointX, pointY, x, y);
      var d = map(d, 0, distance, 255, 0);
      var tempColor = lerpColor(Color1, Color2, map(abs(d), 255, 0, 0, 1));
      set(x, y, tempColor);
    }
  }
  updatePixels();
}

//this one uses a for loop to draw ellipses each one being 1 pixel smaller than the previous one and with a bit of a closer color to the one on the center
function radialBackground2(Color1, Color2) {
  noStroke();
  var end = Math.sqrt(width*width+height*height);
  for (var i = end; i>0; i--) {
    fill(lerpColor(Color1, Color2, map(i, 1, end, 0, 1)));
    ellipse(width/2, height/2, i, i);
  }
}
